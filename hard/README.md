# 🔴 Hard — DSA Practice

Problems in this folder are **advanced level**, for cracking the highest difficulty rounds in Infosys DSE & SP roles.

## 📌 Topics

- Advanced Dynamic Programming (Bitmask DP, Interval DP)
- Complex Graph Algorithms (Dijkstra, Bellman-Ford, Floyd-Warshall, Kruskal, Prim)
- Segment Trees & Fenwick Trees
- Tries
- Advanced Backtracking
- Bit Manipulation
- Divide & Conquer (Advanced)
- String Algorithms (KMP, Rabin-Karp)

## 💡 Tips

- Hard problems require **pattern recognition** from medium-level problems.
- Never brute-force first — always reason about optimal substructure.
- Focus on **time complexity** — O(n log n) or better is usually expected.
- These problems separate **SP-level** candidates from the rest!
