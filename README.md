# Infosys Prep — SP & DSE

DSA practice repository for **Infosys Specialist Programmer (SP)** and **Digital Specialist Engineer (DSE)** roles.

> 📌 **All questions in this repository are categorized Previous Year Questions (PYQs)** based on the **most repeating and important topics** asked in Infosys SP & DSE assessments. Each problem has been handpicked to maximize your preparation efficiency.
